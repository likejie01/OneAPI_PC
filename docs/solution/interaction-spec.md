# 交互规范

## 1. 登录

- 输入账号密码后点击登录
- 失败时原位提示
- 成功后自动进入聊天页

## 2. 聊天

- 选择 AI 后直接进入可用状态
- 回车发送
- 支持停止生成
- 支持重试

## 3. 助手

- 新建时默认空白模板
- 编辑时保留原内容
- 删除需二次确认

## 4. 订阅与钱包

- 套餐可直接用钱包购买
- 支持查看已有订阅与用量
- 购买前显示确认信息

## 5. 我的

- 新建 key 前必须验证密码
- 查看 key 前必须验证密码
- 验证后 30 分钟内免重复验证
- 到期后再次要求验证

## 6. Codex / Claude

- 检测安装状态
- 未安装时展示安装引导
- 安装过程展示进度
- 写配置后必须测试可用性
