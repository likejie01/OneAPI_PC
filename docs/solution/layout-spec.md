# 布局规范

## 1. 总体布局

- 顶部功能栏固定
- 左侧边栏固定，可折叠
- 主内容区自适应
- 内容区内部采用卡片化信息层级

## 2. 桌面壳结构

### 顶部区域

- 左侧显示产品名和当前功能
- 中间显示顶部三功能切换
- 右侧显示当前用户、通知、设置入口

### 左侧边栏

- 助手
- 订阅
- 钱包
- 用量
- 我的
- 可折叠
- 折叠后仅显示图标与悬浮提示

### 内容区域

- 默认采用两栏或三栏结构
- 聊天与工作台优先全宽
- 业务页优先卡片流布局

## 3. 页面宽度

- 桌面主容器最大宽度固定一致
- 列表页允许宽屏展示
- 聊天页采用左历史、右对话或单栏对话

## 4. 响应式规则

- 1200px 以上显示完整边栏
- 960px 到 1200px 可缩窄边栏
- 960px 以下边栏折叠为抽屉

## 5. 视觉规范

- 圆角统一
- 阴影克制
- 边框轻量
- 标题层级清晰
- 中文字体优先系统默认可读字体
