# 页面规范

## 1. 登录页

- 账号、密码、登录按钮
- 支持错误提示
- 成功后进入主框架

## 2. 聊天页

- 模型/AI 列表
- 会话列表
- 消息展示区
- 输入区
- 助手快捷入口

## 3. 助手页

- 助手列表
- 新建助手
- 编辑助手
- 删除助手
- 提示词、模型、参数配置

## 4. 订阅页

- 套餐列表
- 当前订阅
- 已用量
- 购买入口

## 5. 钱包页

- 钱包余额
- 充值入口
- 交易记录
- 页头外内容区展示

## 6. 用量页

- 消耗分布
- 模型调用分析
- 时间区间切换
- 汇总统计

## 7. 我的页

- 个人信息
- Key 列表
- 新建 Key
- 查看 Key
- 复制 Key
- 密码验证

## 8. Codex 页

- 项目选择
- 历史记录
- 消息输入
- 安装状态
- 配置状态

## 9. Claude 页

- 项目选择
